# Minesweeper - Webpack & Vite

A study project comparing two bundlers: Webpack & Vite.
Classic game built with pure typescript.

## About

Classic minesweeper:

- 9x9 size, 10 mines
- LBM - open cell
- RBM - add/remove flag
- Auto open empty cells
- Mines counter

## Getting started

### Webpack

```bash
cd webpack
npm install
npm start
```

App opens at http://localhost:5000

### Vite

```bash
cd vite
npm install
npm run dev
```

App opens at http://localhost:5173

### Production build (webpack)

```bash
npm run build:prod
```
